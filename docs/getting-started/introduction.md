Welcome to Rivia Documentation! This site provides comprehensive documentation for our production-ready Terraform modules and infrastructure best practices.

Rivia provides production-ready Terraform modules for Azure, AWS, and GCP with security best practices and compliance frameworks built-in.

- **Production-Ready Modules**: Battle-tested Terraform modules for common infrastructure patterns
- **Security First**: CIS benchmark compliant with automated security scanning
- **CI/CD Ready**: Pre-configured GitHub Actions workflows for validation and testing
- **Comprehensive Documentation**: Detailed guides, examples, and reference documentation
